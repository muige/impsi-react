# Imperial-SI Unit Converter

A simple, instant unit converter between Imperial and Metric systems. Type a number once and see all conversions instantly.

## Features

- 18 conversion functions across 4 categories:
  - Distance/Length (feet, yards, miles, inches ↔ meters, kilometers, centimeters)
  - Weight/Mass (pounds, ounces ↔ kilograms, grams)
  - Volume (gallons, fluid ounces ↔ liters, milliliters)
  - Temperature (Fahrenheit ↔ Celsius)
- Instant conversions as you type
- Dark/Light theme support (follows system preferences)
- No button clicks required

## Installation

```bash
npm install
```

## Development
